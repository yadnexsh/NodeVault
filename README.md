# Node Vault

A lightweight PySide6 tool to organize, browse, and submit Nuke gizmos, scripts, and templates in a shared environment.

---

## Why this exists

Managing gizmos across artists usually turns into chaos  
Random folders, no previews, no metadata, no consistency  

Node Vault fixes that by providing  
- Structured library browsing  
- Standardized submission flow  
- Centralized asset visibility  

---

## Features

### Library
- Category based navigation
- Nested gizmo grouping (Deep, Image, Draw, etc.)
- Thumbnail grid view
- Adjustable layout (rows and columns)

### Submit
- Upload gizmos, scripts, templates
- Add metadata (author, version, tagline)
- Attach README and extra docs
- Add preview images and demo video
- Tag render type and Nuke version
- Add external links (repo, issues, website)
